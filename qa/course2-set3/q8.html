ユーザーが Manager ロールを持たない場合、状況を正しく表す選択肢は？（左=Container、右=If）

  • A両方のボタンが実行時に表示され、生成 HTML の一部である。
  • Bどちらも実行時に表示されないが、左（Container）ではボタンは生成 HTML の一部である。
  • Cどちらも実行時に表示されないが、右（If）ではボタンは生成 HTML の一部である。
  • Dどちらも実行時に表示されず、生成 HTML の一部でもない。

正解：B

Container は Visible=False でも要素自体は HTML に出力され CSS で非表示になるだけ。If は False 分岐だと要素自体が HTML に生成されません。
