図のアクションはタイマーでトリガーされます。ロジックを10分間実行し続け、その後に新しいタイマーインスタンスを起動したい。Timeout の Assign と条件は？

  • AAssign Timeout = AddMinutes(CurrDateTime(),10) / 条件 Timeout < CurrDateTime()
  • BAssign Timeout += 10 / 条件 Timeout < CurrDateTime()
  • CAssign Timeout += 10 / 条件 CurrDateTime() = Timeout
  • DAssign Timeout = AddMinutes(CurrDateTime(),10) / 条件 CurrDateTime() >= Timeout

正解：D

開始時に Assign で Timeout = AddMinutes(CurrDateTime(), 10) を設定し、ループ内の条件 CurrDateTime() >= Timeout が真になった時点（10分経過）で Timer を再起動して終了します。境界を含む >= 比較なので、ちょうど10分経過した瞬間も確実に検出できます。
