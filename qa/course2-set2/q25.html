次のエンティティ図で、国に未割当か住所に未参照の都市を定期的に削除します。最も高性能なアプローチは？

  • AAggregateで削除対象の都市IDリストを取得し、1つの DELETE を含む Advanced SQL に入力する。
  • BAggregateで対象リストを取得し、ループで deleteCity エンティティアクションを使う。
  • CAdvanced SQL で対象IDを選択し、ループで deleteCity エンティティアクションを使う。
  • D単一の Advanced SQL 要素で DELETE 文を実行する。

正解：D

単一の Advanced SQL で一括 DELETE するのが最も高性能。Aggregate→ループ削除は行ごとのラウンドトリップが発生し非効率です。
