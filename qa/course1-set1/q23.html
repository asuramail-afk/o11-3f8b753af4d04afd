図のエンティティ図で、国に未割当か住所に未参照の都市を定期的に削除します。最も高性能なアプローチは？

  • AAggregate で削除対象の都市 ID リストを取得し、それを 1 つの DELETE を含む Advanced SQL に入力する。
  • B単一の Advanced SQL 要素で DELETE 命令を実行する。
  • CAggregate で対象リストを取得し、ループで DeleteCity エンティティアクションを使う。
  • DAdvanced SQL で対象 ID を選択し、ループで DeleteCity エンティティアクションを使う。

正解：B

単一の Advanced SQL で一括 DELETE するのが最も高性能。Aggregate→ループ削除は行ごとのラウンドトリップが発生し非効率です。
